3) Проверь регистры имён и пути
Имена файлов на Replit/Node чувствительны к регистру. Файл stats-cards.tsx не равен Stats-Cards.tsx.

Убедись, что tsconfig.json содержит alias @/*. Если нет — либо добавь, либо замени @/components/... на относительные пути:

json
Копировать
Редактировать
{
  "compilerOptions": {
    "baseUrl": ".",
    "paths": { "@/*": ["./*"] }
  }
}
